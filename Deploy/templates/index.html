Hướng dẫn gọi API

Base URL: http://localhost:8000

POST /predict
cURL
curl -X POST http://localhost:8000/predict \ -H "Content-Type: application/json" \ -d '{"text": "33 người chết ở bệnh viện Chợ Rẫy vì virus corona"}'
POST /debug/preprocess — Xem text được xử lý thế nào
Debug preprocessing
curl -X POST http://localhost:8000/debug/preprocess \ -H "Content-Type: application/json" \ -d '{"text": "33 người chết ở bệnh viện Chợ Rẫy"}'
Python
Python requests
